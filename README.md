# ADHD Detection Using Pose Estimation

This project uses **MediaPipe** and **OpenCV** to analyze body movements and postures for ADHD detection.

## Tech Stack
* **Language:** Python
* **Libraries:** MediaPipe, OpenCV, NumPy, Pandas.
* **Platform:** Google Colab / Jupyter Notebook.

## Project Structure
* `app.py` - Main application script.
* `images/` - Contains result screenshots.
* `requirements.txt` - List of required Python libraries.
* `CHILDREN ADHD DISEASE DETECTION...ipynb` - Main code logic file.
